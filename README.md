# quizzes

A Vite + React + TypeScript app for building a quiz from your own questions.

## What it does

- Add questions and four answer choices per question
- Mark the correct answer in the editor
- Generate a shuffled quiz from the questions you entered
- Take the quiz one question at a time
- Review your score and missed answers at the end

## Run locally

```bash
npm install
npm run dev
```

## Build for production

```bash
npm run build
```
